Vida

Vida is a simple game. Each player (Red and Black) writes a basic JavaScript AI that is loaded into every 'cell' (right now each team starts with 100). The object is to capture all of your enemies' cells. Like in the game Go, you capture an enemy cell by surrounding it on all sides (up, down, left, right). Currently both Red and Black are running an AI called Speeder 3. Let me know if you have any comments or suggestions. The next step is probably to set up a better demo that allows you to choose from a few different AI. The code is on GitHub.
